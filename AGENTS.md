# AGENTS 协作规则

## 产品文档目录约定

所有产品文档统一放在：
- `F:\Project\My-Project\Agent\actionAgent\docs`

目录职责划分：
- `docs/app`：客户端文档
- `docs/agent`：Agent 核心模块文档
- `docs/webui`：WebUI 文档

产品级宏观 PRD：
- `docs/PRD.md`：整个产品的宏观描述文档

协作要求：
- 新增或修改产品文档时，必须按上述职责放入对应目录。
- 未经明确说明，不在 `docs` 目录之外存放产品文档。
- 修改 `README.md` 时，必须同步更新 `README.zh-CN.md`；修改 `README.zh-CN.md` 时，也必须同步检查并更新 `README.md`，保持中英文产品说明一致。
